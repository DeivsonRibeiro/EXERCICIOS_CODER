# 💻 Exercícios Coder

## 📌 Sobre o Projeto

Repositório contendo diversos exercícios práticos desenvolvidos durante estudos de programação. O objetivo é consolidar conceitos fundamentais como lógica, manipulação de dados e estruturação de código.

## 🛠️ Tecnologias Utilizadas

* JavaScript
* HTML
* CSS

## ⚙️ Conteúdos Praticados

* Estruturas condicionais
* Laços de repetição
* Funções
* Manipulação de dados
* Lógica de programação

## 🎯 Objetivo

Este projeto foi criado com foco em aprendizado e evolução contínua, servindo como base para projetos mais complexos.

## 📚 Aprendizados

* Organização de código
* Boas práticas iniciais
* Resolução de problemas
* Pensamento lógico

## ▶️ Como Executar

```bash
# Clone o repositório
git clone https://github.com/DeivsonRibeiro/EXERCICIOS_CODER.git
```
